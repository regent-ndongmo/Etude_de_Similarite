# Analyse avancee de similarite des etudes

## Lecture recommandee

- Lire d'abord `README_EXECUTION.md` pour installer et relancer le projet
- Ouvrir ensuite `rapport_interpretation.md` pour comprendre les sorties
- Consulter `top_25_paires_similaires.csv` pour les paires les plus proches

## Fichiers principaux

- `analyse_similarite_etudes_avancee.py` : script principal
- `requirements.txt` : dependances versionnees
- `README_EXECUTION.md` : guide d'installation et d'execution
- `paires_similaires_detaillees.csv` : detail complet des paires
- `clusters_etudes.csv` : regroupement des etudes par proximite
- `matrice_similarite_0_100.csv` : matrice complete des scores
